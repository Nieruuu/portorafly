A bit about
me

Saya Rafly — Mahasiswa semester 8 Sistem Informasi yang suka membuat projek untuk menyelesaikan masalah nyata. Perjalanan saya dimulai dari keingintahuan tentang bagaimana website bekerja, dan berkembang menjadi kecintaan pada fullstack development dan DevOps.

Selama magang di perusahaan yang berkolaborasi dengan pemerintah, saya mendapat kesempatan langsung menangani proyek nyata — mulai dari membangun automation pipeline menggunakan Python dan Docker, hingga mengembangkan Taxation Map dengan React untuk visualisasi data perpajakan pada peta interaktif.

Saat ini saya terus mengasah kemampuan di Laravel, React, Livewire dan ekosistem DevOps — selalu mencari cara yang lebih efisien dan clean dalam membangun sesuatu.
